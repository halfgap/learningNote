Box（盒子） 模型

盒子模型：页面元素绘制时候其占用空间方式

案例：

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>盒子模型</title>
<style type="text/css">

#box{
    margin: 8px; /* 设定外边距，盒子与盒子之间的距离 */
}
#box2{
    /* id选择器 优先于 类选择器 */
    border: 3px dashed #00f; /* 设定边框，盒子皮的厚度 */
    /* margin: 18px; */ /* 对四个边同时设置外边距 */
    /* margin-left: 8px;*/  /* 单独设置左侧外边距 */ 
    /* margin-right: 8px; */
    /* margin-top: 18px; */
    /* margin-bottom: 8px; */

    /* margin: 上 右 下 左; 单行设置4边属性，从后向前
    可以省略数值，如果省略则使用对边代替*/
    margin: 18px 8px 8px ;
}
.demo{
    /* 利用类选择器复用样式 */ 
    width: 100px; /* 设定内容区域的宽 */
    height: 80px; /* 设定内容区域的高 */
    padding: 20px;/* 设定内边距，内容与盒子边界的缝隙 */
    border: 3px dashed #ddd; /* 设定边框，盒子皮的厚度 */
    /*背景绘制在内边距之内，外边距范围不绘制背景*/   
    background-color: #ff0;
}
</style>
</head>
<body>
    <h1>盒子模型</h1>
    <div id="box" class="demo">范传奇呵呵</div>
    <div id="box2" class="demo">盒子外边距相互侵占</div>
</body>
</html>

盒子模型中的内容

一般情况下，如果不设置宽高，盒子随着内容自动增长。当设置了宽高高以后，如果内容超出盒子的范围，可以利用溢出属性设置内容的显示方式

案例：

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>溢出内容属性</title>
<style type="text/css">
.c1, .c2, .c3, .c4{
    border: 1px solid #00f; 
    margin: 10px;
}
.c2, .c3, .c4{
    width: 100px;
    height: 80px;
}
.c3{
    /* 溢出盒子边界的内容隐藏不显示 */ 
    overflow: hidden;
}
.c4{
    overflow: scroll; /* 显示滚动条，无论内容是否溢出 */
    /*overflow: auto;*/ /*根据内容自动控制是否显示滚动
    条， 如果内容没有溢出就不显示滚动条，否则显示*/
}
.c5{
    width: 200px;
    height: 50px;
    border: 1px solid #eee;
    overflow: auto;
}
</style>
</head>
<body>
    <h1>内容溢出属性</h1>

    <div class="c5"><img src="../k.jpg"></div>

    <div class="c4">c4:设置溢出隐藏后，多出的内容不再显示：
    范传奇这个先生每天早晨都会在
    一个固定时间，干一件大事！</div> 
    <div class="c3">c3:设置溢出隐藏后，多出的内容不再显示：
    范传奇这个先生每天早晨都会在
    一个固定时间，干一件大事！</div> 
    <div class="c2">c2:设置大小以后内容仍然显示：
    范传奇这个先生每天早晨都会在
    一个固定时间，干一件大事！</div> 
    <div class="c1">c1:默认情况下盒子大小随着内容自动增长
    范传奇这个先生每天早晨都会在
    一个固定时间，干一件大事！</div>
</body>
</html>

背景设置

设置盒子模型的 border内部的背景绘制效果。

  1. 设置背景色
  2. 设置背景图片
    • 设置图片的重复方式，平铺，x方向重复，y方向重复
    • 设置图片的偏移位置
      • 相对位置偏移
      • 绝对位置偏移

元素的显示方式

有3种： - 块： div, h1 ~ h6, p 等 - 独立占有行，可以设置宽高 - 行内：span b i u label 等 - 不独立占有行，和其它文本混合在一行，不能设置宽高！ - 行内块 - 不独立占有行，和其它文本混合在一行，能够设置宽高！

利用CSS可以更改元素的显示方式，从行内元素显示方式更改为块（行内块）元素，为了设置元素的宽高，进行布局占位。

定位

如何设置元素的显示位置，称为定位

  • 流定位（浏览器中元素的默认定位）:static
    • 块元素流定位：从上到下依次显示
    • 行内元素流定位：从左到右依次显示
  • 浮动定位:float
  • 相对定位:relative
  • 绝对定位:absolute
  • 固定定位:fixed

利用CSS更改元素定位属性，实现调整元素的显示位置的目的。
