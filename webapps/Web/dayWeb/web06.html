JavaScript

  • html搭建页面的结构和内容
  • css用于美化页面

  • javaScript，给页面添加动态效果和内容的

  • 历史：1995年由网景公司发布 LiveScript同年更名为JavaScript简称JS，和java没啥关系 为了蹭热度。

JS特点

  • js属于脚本语言，不需要编译，由浏览器解析执行。
  • js可以嵌入到html代码中
  • js基于面向对象，属于弱类型语言

JS优点

  • 交互性：可以直接和用户进行交互
  • 安全性：js语言智能访问浏览器内部的数据不能访问浏览器之外内容（电脑磁盘中的各种数据）

如何引入JS代码

  1. 在元素的事件中书写js代码

  2. 通过Script标签添加js代码

    <script type="text/javascript">
    alert("第二种引入方式"); 
</script>
  3. 把js代码写在单独的js文件中，通过标签的src属性 引入js文件 <script type="text/javascript" src="first.js"></script>

JS的数据类型

  • 数值类型(number)
  • 字符串类型(string)
  • 布尔值类型(boolean)
  • undefined
  • 对象类型

变量的声明及赋值

  • 由于js语言属于弱类型语言声明变量时不需要指定类型 var x = 18; var name="小明"; var isBoy=true; x="abc"; var m; //此时m的类型为undefined值也为undefined
